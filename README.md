# 🔴 Uptime Notifier

A Python script that checks if a website is up and sends an email alert if it’s down.

## ⚙️ Features
- Checks your website status (200 OK)
- Sends alert email if:
  - Site is down
  - Timeout or error occurs
- Easy to automate using Task Scheduler

## 🔧 Tech Used
- Python
- requests (for HTTP)
- smtplib (for email)
- Task Scheduler (for automation)
